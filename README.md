## Getting Started

Basic guide to getting Finish-em up and running

### Installing

Initially clone the repo

```
git clone https://github.com/ryankscott/finish-em.git
```

Install the dependencies

```
npm install
```

## Developing

### Running the dev server

```
npm run start
```
