# TUI Calendar Date Picker

## Summary

Define the interactive terminal calendar picker for visual date selection in TUI date-editing flows.

## Current Rules

- TUI provides a calendar picker for date fields: The TUI SHALL provide an interactive terminal calendar picker that allows users to select a date visually using keyboard navigation when editing any date field (due date, scheduled date, project start date, project end date).
- Calendar picker hint text is shown in field pickers: The TUI MUST display a hint to users that the calendar picker is available when a date field is selected for editing.
