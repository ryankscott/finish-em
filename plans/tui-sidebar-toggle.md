# TUI Sidebar Toggle

## Summary

Define keyboard-driven sidebar (project list and nav) visibility behavior in the TUI.

## Current Rules

- Sidebar visibility can be toggled via keyboard shortcut: The TUI SHALL provide a dedicated keyboard shortcut that toggles the sidebar (project list and nav) between visible and collapsed states from any normal interaction context (when the user is not in text input mode or a modal/picker).
