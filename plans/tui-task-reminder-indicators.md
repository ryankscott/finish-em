# TUI Task Reminder Indicators

## Summary

Define reminder presence and timing indicators shown on task rows in the TUI task list.

## Current Rules

- Task rows show reminder presence in collapsed state: The TUI task list SHALL render reminder presence on collapsed task rows with a bell indicator.
- Expanded task rows show reminder time details: The TUI task list SHALL render reminder timing details on expanded task rows.
